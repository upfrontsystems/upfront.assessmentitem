upfront.assessmentitem Installation
------------------------

To install upfront.assessmentitem using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``upfront.assessmentitem`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        upfront.assessmentitem
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
